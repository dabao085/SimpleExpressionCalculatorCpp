# 一个简单的自己实现的C++表达式计算器小程序  
C++练习，表达式计算器，使用类的继承。
题目出处: **《Linux多线程服务端编程-使用muduo C++网络库》** P.584。

## 输入(input)  
仅支持`+`, `-`, `*`, `/`，暂不支持括号提高优先级  
例如:  
`2 * 5 + 1`
## 输出(output)  
`2 * 5 + 1 = 11`
